  • class Académica "Sistema de votación en linea"

    Desarrollador Web

    Agosto 2017 - Noviembre 2017

    • Trabaje con mi equipo usando la metodología ágil Scrum siguiendo correctamente las etapas para tener un resultado favorable.
    • El objetivo del sistema era llevar un conteo de votos para el TSE que permitiera la elección de los 84 diputados para la asamblea legislativa mostrando reportes en PDF y gráficos interactivos en tiempo real de las elecciones.
    • Manteniendo un estricto grado de seguridad para el votante incluido las validaciones como: Si el usuario no está registrado no tiene derecho a ejercer su voto, y cuando este ejercía su voto el sistema no permitía que lo hiciera nuevamente.
    • Almacenar los votos, votante, candidatos y partidos en la DB.
    • Además, el sistema proporcionó módulos CRUD para administrar los datos de la DB.
